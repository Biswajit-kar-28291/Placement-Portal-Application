# 🚀 Placement Portal Application

A full-stack web application designed to streamline the campus placement process for students and recruiters. This system allows students to apply for jobs and companies to manage hiring efficiently.

👉 [Watch Full Demo Video](https://drive.google.com/file/d/1dL0RNMQvNhgUORzcW6lrEk-XamTtYgA-/view?usp=drivesdk )

---

## 📌 Features

- 👨‍🎓 Student Registration & Login  
- 🏢 Company Registration & Login  
- 📄 Resume Upload System  
- 📢 Job/Placement Drive Posting  
- 📝 Apply for Jobs  
- 📊 Application Status Tracking  
- 🚫 Prevent Duplicate Applications  
- 🔐 Role-Based Authentication System  
- 💻 Responsive UI using Bootstrap  

---

## 🛠 Tech Stack

- **Backend:** Python, Flask  
- **Frontend:** HTML, CSS, Bootstrap, Jinja2  
- **Database:** SQLite  
- **ORM:** SQLAlchemy
